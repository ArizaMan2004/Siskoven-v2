# Guía Rápida - Venko

## Instalación Rápida (5 minutos)

\`\`\`bash
# 1. Instalar dependencias
npm install

# 2. Crear .env.local con credenciales de Firebase
cp .env.example .env.local
# Edita .env.local y agrega tus credenciales

# 3. Ejecutar
npm run dev

# 4. Abre http://localhost:3000
\`\`\`

## Primeros Pasos

1. **Registrarse** → Email + Contraseña
2. **Agregar Productos** → Nombre, Precio USD, Ganancia %
3. **Actualizar Tasa BCV** → Ingresa tasa diaria
4. **Realizar Venta** → Busca producto, agrega cantidad, paga
5. **Ver Reportes** → Descarga PDF

## Atajos Útiles

- **Buscar Producto**: Escribe en el campo de búsqueda
- **Escanear Código**: Presiona cualquier tecla para activar escáner
- **Actualizar Precios**: Cambia la tasa BCV
- **Descargar Reporte**: Ve a Reportes → Selecciona tipo → Descarga

## Métodos de Pago

- Efectivo
- Tarjeta
- Transferencia
- Mixto (Efectivo + Tarjeta)

## Tipos de Productos

- **Unidad**: Se vende por cantidad (ej: 5 unidades)
- **Peso**: Se vende por kg (ej: 2.5 kg)
